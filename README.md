# 📿 Tasbeeh.js - Aesthetic Digital Tasbeeh

A minimalist, aesthetic, and functional Digital Tasbeeh web application. Designed with a premium **Emerald Green** theme and optimized for a seamless mobile web app experience.

## ✨ Features

- **Custom Goals**: Choose your target dhikr (33, 100, or 1000).
- **Daily History Tracking**: Automatically saves your completed dhikr sessions to a history list.
- **Dynamic Input**: Customize the name of the dhikr you are reciting.
- **Progress Feedback**: Haptic feedback (vibration) on each click (mobile only) and target alerts.
- **Premium Design**: Emerald Green aesthetic with glassmorphism effects and spiritual background.
- **Mobile Optimized**: Add to Home Screen support for a native app-like experience.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom styling with Flexbox and Glassmorphism.
- **Vanilla JavaScript**: Logic for counting, history tracking, and DOM manipulation.
- **Google Fonts**: Poppins for a modern typography.

## 📱 Installation (PWA Style)

You can use this as an app on your smartphone:
1. Open the [Live Demo](https://bukanwani68.github.io/Tasbeeh.js/) in your mobile browser.
2. Tap the **Share** (iOS) or **Menu** (Android) icon.
3. Select **"Add to Home Screen"**.
4. The Tasbeeh icon will now appear on your app drawer!
